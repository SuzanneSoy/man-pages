名前

dh_lintian - ビルドディレクトリに lintian への override ファイルをインストールする  

書式

dh_lintian [debhelper options]  

説明

dh_lintian is a debhelper program that is responsible for installing override files used by lintian into package build directories.  

ファイル

debian/package.lintian-overrides
パッケージビルドディレクトリ以下の usr/share/lintian/overrides/package へインストールを行います。このファイルは、lintian が誤った診断をするのを防ぐ為に利用されます。
debian/source/lintian-overrides
これらファイルはインストールされません。しかしながら、ソースパッケージに対する診断結果を再定義する為、lintian にて本ファイルの中身がスキャンされます。
 

参照

debhelper(1)
