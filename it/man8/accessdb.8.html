NAME

accessdb - mostra il contenuto di una base di dati man-db in un formato comprensibile.  

SINTASSI

/usr/sbin/accessdb [<file-index>]  

DESCRIZIONE

accessdb mostretà i dati contenuti all'interno di una base di dati di man-db in un formato comprensibile (all'uomo :). Di default, mosterà i dati di /var/cache/man/index.<tipo-db>, dove <tipo-db> dipende dalla libreria di basi di dati in uso.

Fornendo un argomento ad accessdb di modificherà questo comportamento predefinito.
