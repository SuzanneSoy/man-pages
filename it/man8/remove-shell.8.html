NOME

remove-shell - rimuove shell dall'elenco di quelle valide per il login  

SINTASSI

remove-shell nomeshell [nomeshell...]  

DESCRIZIONE

remove-shell opera sui file temporanei /etc/shells.tmp e /etc/shells.tmp2 per rimuovere le shell specificate dall'elenco delle shell di login valide e copia di nuovo il risultato in /etc/shells.
