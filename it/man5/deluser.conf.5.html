NOME

/etc/deluser.conf - file di configurazione per deluser(8) e delgroup(8).  

DESCRIZIONE

Il file /etc/deluser.conf contiene le impostazioni predefinite per i programmi deluser(8) e delgroup(8). Tutte le opzioni hanno il formato opzione = valore ed è consentito racchiudere i valori fra virgolette (") o apici ('). Le righe di commento devono iniziare con il carattere cancelletto (#).

deluser(8) e delgroup(8) leggono anche /etc/adduser.conf, si veda adduser.conf(5); le impostazioni in deluser.conf possono prevalere sulle impostazioni in adduser.conf.

Le opzioni di configurazione valide sono:

REMOVE_HOME
Cancella la directory home e lo spool di posta dell'utente da rimuovere. Il valore può essere 0 (non cancella) o 1 (cancella).
REMOVE_ALL_FILES
Cancella tutti i file sul sistema che appartengono all'utente da rimuovere. Quando questa opzione è attiva REMOVE_HOME non ha effetto. Il valore può essere 0 o 1.
BACKUP
Se REMOVE_HOME o REMOVE_ALL_FILES sono attivi allora viene fatto il backup di tutti i file prima della loro cancellazione. Il file con il backup viene chiamato nomeutente.tar(.gz|.bz2) e viene scritto nella directory specificata dall'opzione BACKUP_TO. Il metodo di compressione scelto è il migliore fra quelli disponibili. I valori validi sono 0 e 1.
BACKUP_TO
Quando è attivo BACKUP, BACKUP_TO specifica in quale directory viene scritto il backup. Il valore predefinito è la directory corrente.
NO_DEL_PATHS
Un elenco di espressioni regolari separate da spazi. Ogni file da cancellare perché contenuto all'interno della directory home o perché appartenente all'utente da rimuovere viene confrontato con queste espressioni regolari e, se corrisponde, non viene cancellato. Il valore predefinito è l'elenco delle directory di sistema, esclusa /home.

In altre parole: con la configurazione predefinita saranno cancellati solo i file all'interno di /home che appartengono all'utente specificato.

ONLY_IF_EMPTY
Rimuove un gruppo solo se non ci sono utenti appartenenti a tale gruppo. Il valore predefinito è 0.
EXCLUDE_FSTYPES
Una espressione regolare che indica quali file system devono essere esclusi durante la ricerca dei file dell'utente da eliminare. Il valore predefinito è "(proc|sysfs|usbfs|devpts|tmpfs|afs)".
