NOME

deb-extra-override - file override extra degli archivi Debian  

SINTASSI

override  

DESCRIZIONE

Sebbene la maggior parte delle informazioni su un pacchetto binario/sorgente possano essere trovate nel file control/.dsc, tutte possono essere sovrascritte al momento dell'esportazione nei file Packages/Sources. Il file override extra contiene questi valori da sovrascrivere.

Il file override extra ha un semplice formato delimitato da spazi bianchi. Sono permessi i commenti, indicati con un carattere #.

pacchetto nome-campo valore

pacchetto è il nome del pacchetto binario/sorgente.

nome-campo è il nome del campo che viene sovrascritto. valore è il valore da inserire nel campo. Può contenere spazi dato che la riga non viene suddivisa in più di 3 colonne quando viene analizzata.

I file override extra usati per creare i file elenco Packages ufficiali possono essere trovati nella directory indices di ogni mirror Debian.  

VEDERE ANCHE

dpkg-scanpackages(1), dpkg-scansources(1), apt-ftparchive(1).
