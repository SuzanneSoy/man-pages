NAME

xfce4-popup-whiskermenu - shows Whisker Menu

 

SYNOPSIS

xfce4-popup-whiskermenu -h
xfce4-popup-whiskermenu --help
xfce4-popup-whiskermenu -V
xfce4-popup-whiskermenu --version
xfce4-popup-whiskermenu [ -p ]

 

DESCRIPTION

xfce4-popup-whiskermenu shows Whisker Menu if at least one Whisker Menu applet is present on one of your panels.

The default behavior is to show Whisker Menu near the applet button. If you have several Whisker Menu applets, only the first one you have set up will pop up.

 

OPTIONS

-h, --help
Show help options.
-p, --pointer
Popup menu at current mouse position.
-V, --version
Print version information and exit.
