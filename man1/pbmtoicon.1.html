NAME

pbmtoicon - convert a portable bitmap into a Sun icon  

SYNOPSIS

pbmtoicon [pbmfile]  

DESCRIPTION

Reads a portable bitmap as input. Produces a Sun icon as output.  

SEE ALSO

icontopbm(1), pbm(5)
