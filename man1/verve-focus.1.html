NAME

verve-focus - Focus the xfce4-verve-plugin input area  

SYNOPSIS

verve-focus  

DESCRIPTION

Running this command focuses the xfce4-verve-plugin's input area from the commandline. It may be especially useful to bind to a menu item or a key press while using the xfce4-verve-plugin.
