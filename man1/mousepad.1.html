NAME

mousepad --- simple text editor for Xfce  

SYNOPSIS

mousepad [files]
   

DESCRIPTION

This manual page documents briefly the mousepad command.

mousepad is a simple text editor for the Xfce Desktop Environment.  

Usage:

mousepad [OPTION...] [FILES...]
 

Help Options:

-h, --help
Show help options
--help-all
Show all help options
--help-gtk
Show GTK+ Options

GTK+ Options

--class=CLASS
Program class as used by the window manager
--name=NAME
Program name as used by the window manager
--screen=SCREEN
X screen to use
--sync
Make X calls synchronous
--gtk-module=MODULES
Load additional GTK+ modules
--g-fatal-warnings
Make all warnings fatal
 

Application Options:

--disable-server
Do not register with the D-BUS session message bus
-q, --quit
Quit a running Mousepad instance
-v, --version
Print version information and exit
--display=DISPLAY
X display to use
