NAME

scdoc - generate man(7) manual pages

 

SYNOPSIS

scdoc < input

 

DESCRIPTION

The scdoc utility reads scdoc(5) syntax from the standard input and writes man(7) style roff to the standard output.

 

SEE ALSO

scdoc(5)
