NAME

fwupdtpmevlog - Debugging utility for the TPM event log  

DESCRIPTION

 

Usage:

fwupdtpmevlog [OPTION?]
 

Help Options:

-h, --help
Show help options
 

Application Options:

-v, --verbose
Show extra debugging information
-p, --pcr
Only show single PCR value

This tool will read and parse the TPM event log from the system firwmare.
