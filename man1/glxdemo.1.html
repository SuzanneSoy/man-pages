NAME

glxdemo - a demonstration of the GLX functions  

SYNOPSIS

glxdemo  

DESCRIPTION

The glxdemo program shows how to use the GLX functions in order to create an OpenGL program running on top of an X server. This program is most useful when studied in its source code form.  

AUTHOR
