hman - πλοήγηση στις ηλεκτρονικές σελίδες εγχειριδίου  

ΣΥΝΟΨΗ

hman [ -P φυλλομετρητής ] [ -H υπολογιστής_υπηρεσίσς ] [ ενότητα ] όνομα
hman [ -P φυλλομετρητής ] [ -H υπολογιστής_υπηρεσίας ] [ ενότητα ] [ ευρετήριο ]  

ΠΕΡΙΓΡΑΦΗ

Το σενάριο hman είναι μιά διεπαφή προς τη man2html(1) που σου επιτρέπει να εισάγεις αιτήσεις σελίδων εγχειριδίου στη γραμμή διαταγών, και να βλέπεις την έξοδο στο φυλλομετρητή της προτίμησής σου. Η συμπεριφορά θυμίζει τη man(1) ώστε πολλοί μπορεί να θελήσουν να δώσουν στην hman το ψευδώνημο man. Αν ο εν χρήσει φυλλομετρητής είναι ο netscape, και αν τρέχει ήδη, τότε το σενάριο hman θα μεταβιβάσει την αίτηση σ' αυτόν.

 

ΕΠΙΛΟΓΕΣ

-P φυλλομετρητής
Προσδιορίζει το φυλλομετρητή (όπως lynx, xmosaic, arena, chimera, netscape, amaya, ...) που θα χρησιμοποιηθεί. Αυτή η επιλογή υπερισχύει της μεταβλητής περιβάλλοντος MANHTMLPAGER. Η προεπιλογή είναι η μη-httpd έκδοση του lynx.
-H υπολογιστής_υπηρεσίας
Προσδιορίζει από ποιόν υπολογιστή θα πάρει τις σελίδες εγχειριδίου. Αυτή η επιλογή υπερισχύει της μεταβλητής περιβάλλοντος MANHTMLHOST. Η προεπιλογή είναι localhost.

 

ΠΕΡΙΒΑΛΛΟΝ

MANHTMLPAGER
Ο προεπιλεγόμενος φυλλομετρητής, προκύπτει από τη χρήση αυτής της μεταβλητής περιβάλλοντος.
MANHTMLHOST
Ο προεπιλεγόμενος υπολογιστής υπηρεσίας, προκύπτει από τη χρήση αυτής της μεταβλητής περιβάλλοντος.
