NAME

FcLangNormalize - Normalize the language string  

SYNOPSIS

#include <fontconfig/fontconfig.h>

FcChar8 * FcLangNormalize (const FcChar8 *lang);  

DESCRIPTION

Returns a string to make lang suitable on fontconfig.
