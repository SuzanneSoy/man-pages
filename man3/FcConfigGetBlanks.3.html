NAME

FcConfigGetBlanks - Get config blanks  

SYNOPSIS

#include <fontconfig/fontconfig.h>

FcBlanks * FcConfigGetBlanks (FcConfig *config);  

DESCRIPTION

FcBlanks is deprecated. This function always returns NULL.
