NAME

cpow, cpowf, cpowl - complex power function  

SYNOPSIS

#include <complex.h>

double complex cpow(double complex x, complex double z);
float complex cpowf(float complex x, complex float z);
long double complex cpowl(long double complex x,
                          complex long double z);

Link with -lm.
 

DESCRIPTION

These functions calculate x raised to the power z (with a branch cut for x along the negative real axis.)  

VERSIONS

These functions first appeared in glibc in version 2.1.
