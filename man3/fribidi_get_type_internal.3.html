NAME

fribidi_get_type_internal - get character bidi type  

SYNOPSIS

#include <fribidi.h>

extern FriBidiCharType fribidi_get_type_internal(FriBidiChar ch);  

PARAMETERS

FriBidiChar ch
Input character.
 

DESCRIPTION

Deprecated. Replaced by fribidi_get_bidi_type.  

RETURNS

Since there are 23 possible values for an extern FriBidiCharType, they are not all listed here.
