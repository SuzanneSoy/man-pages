NAME

FcPatternHash - Compute a pattern hash value  

SYNOPSIS

#include <fontconfig/fontconfig.h>

FcChar32 FcPatternHash (const FcPattern *p);  

DESCRIPTION

Returns a 32-bit number which is the same for any two patterns which are equal.
