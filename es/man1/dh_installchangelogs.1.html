NOMBRE

dh_installchangelogs - Instala los ficheros de cambios en los directorios de construcción  

SINOPSIS

dh_installchangelogs [opciones-de-debhelper] [-k] [-Xelemento] [fuente-original-de-software]  

DESCRIPCIÓN

dh_installchangelogs es un programa de debhelper responsable de la instalación de los ficheros de registro de cambios en los directorios de construcción del paquete.

An upstream changelog file may be specified as an option.

If there is an upstream changelog file, it will be installed as usr/share/doc/package/changelog in the package build directory.

If the specified upstream changelog is an html file (determined by file extension), it will be installed as usr/share/doc/package/changelog.html instead. If the html changelog is converted to plain text, that variant can be specified as a second upstream changelog file. When no plain text variant is specified, a short usr/share/doc/package/changelog is generated, pointing readers at the html changelog file.  

FICHEROS

debian/changelog
debian/NEWS
debian/paquete.changelog
debian/paquete.NEWS
Automáticamente instalado en «usr/share/doc/package/» en el directorio de construcción del paquete.

Use el nombre específico del paquete si el paquete necesita un fichero de changelog o NEWS diferente.

El fichero changelog se instala con el nombre «changelog» para los paquetes nativos, y changelog.Debian para paquetes no nativos. El fichero NEWS siempre se instala con el nombre NEWS.Debian.

 

OPCIONES

-k, --keep
Conserva el nombre original del fichero de cambios del desarrollador principal. Esto se realiza instalando el fichero de cambios del desarrollador principal como changelog, y haciendo un enlace simbólico de éste al nombre original del fichero. Puede ser útil si el fichero de cambios del desarrollador principal tiene un nombre poco usual, o si alguna otra documentación en el paquete hace referencia al fichero changelog.
-Xelemento, --exclude=elemento
No se instalarán los ficheros de registro de cambios del desarrollador principal que contengan elemento en alguna parte de su nombre.

Note that directory name of the changelog is also part of the match.

upstream
Instala este fichero como el fichero de cambios del desarrollador principal.
