NOMBRE

debconf-get-selections - Devuelve el contenido de una base de datos de debconf  

SINOPSIS

debconf-get-selections [--installer]  

DESCRIPCIÓN

Devuelve la base de datos de debconf actual en un formato comprensible por «debconf-set-selections».

Use el parámetro «--installer» para volcar la base de datos de debconf de debian-installer, tomando como fuente «/var/log/installer/cdebconf».
