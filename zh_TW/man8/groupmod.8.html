名稱

groupmod - 修 改 群 組  

語法

groupmod [-g gid [-o]] [-n group_name ] group  

描述

groupmod 命 令 會 參 照 你 命 令 列 上 指 定 的 部 份 修 改 系 統 帳 號 檔 。 下 列 為 groupmod 可 選 用 的 參 數 。

-g gid
群 組 I D 值 。 必 須 為 唯 一 的 ID 值 ， 除 非 用 -o 選 項 。 數 字 不 可 為 負 值。預 設 為 最 小 不 得 小 於 999 而 逐 次 增 加 。 0~ 499 傳 統 上 是 保 留 給 系 統 帳 號 使 用 。 如 果 有 檔 案 使 用 舊 的 群 組 ID ， 而 這 時 候 你 新 增 的 群 組 ID 恰 好 與 舊 的 相 同 ， 這 樣 的 話 你 要 手 動 改 一 下 這 些 檔 案 的 群 組 ID 。
-n group_name
更 改 群 組 名 。
 

檔案

/etc/group - 群 組 資 訊
/etc/gshadow - 群 組 資 訊 加 密
