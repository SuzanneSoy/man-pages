名稱

chsh - 更換你登入時所用的shell  

語法

chsh [ -s shell ] [ -l ] [ -u ] [ -v ] [ username ]  

描述

chsh 用來更換你登入時所用的shell 。在命令列沒指定shell。 chsh 會提示一個給你。  

有效的shell

chsh 可用指定路徑全名的方式來指定任何一個可執行的檔案。 然而，如果此 shell並沒紀錄在 /etc/shells 檔上，則chsh會提出警告訊息。  

選項

-l, --list-shells
列出紀錄在 /etc/shells 的shell。
-s, --shell
變更你的登入shell.
-u, --help
使用短訊。
-v, --version
版本資訊
 

相關文件
