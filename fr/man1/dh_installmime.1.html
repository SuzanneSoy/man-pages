NOM

dh_installmime - Installer les fichiers « mime » dans le répertoire de construction du paquet  

SYNOPSIS

dh_installmime [options_de_debhelper]  

DESCRIPTION

dh_installmime est le programme de la suite debhelper chargé de l'installation des fichiers « mime » dans le répertoire de construction du paquet.  

FICHIERS

debian/paquet.mime
Installé dans le répertoire de construction du paquet, sous usr/lib/mime/packages/paquet.
debian/paquet.sharedmimeinfo
Installé dans le répertoire de construction du paquet, sous /usr/share/mime/packages/paquet.xml.
 

VOIR AUSSI

debhelper(7)

Ce programme fait partie de debhelper.
