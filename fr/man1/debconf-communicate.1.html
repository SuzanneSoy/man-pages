NOM

debconf-communicate - communiquer avec debconf  

SYNOPSIS

 echo commandes | debconf-communicate [options] [paquet]
 

DESCRIPTION

debconf-communicate permet de communiquer à la volée avec debconf depuis la ligne de commande. Le paramètre paquet est le nom du paquet que vous voulez faire communiquer avec debconf, il peut être omis si vous êtes paresseux. Ce programme lit depuis l'entrée standard des commandes utilisées par le protocole de debconf. Pour la documentation sur les commandes disponibles ainsi que leur usage, veuillez consulter les spécifications de debconf.

Les commandes sont exécutées dans l'ordre. Le code de retour de chaque commande est affiché sur la sortie standard.

La valeur de retour de ce programme est le code de retour de la dernière commande exécutée.  

EXEMPLE

 echo get debconf/frontend | debconf-communicate

Afficher l'importance de la question de l'interface/debconf.  

ATTENTION

Ce programme ne devrait pas être utilisé par les scripts de maintenance d'un paquet qui utilise debconf ! Il peut, cependant, être utile pour le débogage.
