NOM

dh_listpackages - Énumérer les paquets binaires que debhelper va traiter  

SYNOPSIS

dh_listpackages [options_de_debhelper]  

DESCRIPTION

dh_listpackages est le programme de la suite debhelper chargé de produire la liste de tous les paquets binaires que les commandes debhelper traiteront. Si ce programme reçoit des paramètres, il adapte cette liste afin de la rendre conforme à la liste des paquets qui seraient traités par les autres programmes debhelper s'ils recevaient ces mêmes paramètres.

Les paquets sont indiqués dans le même ordre que dans le fichier debian/control.  

VOIR AUSSI

debhelper(7)

Ce programme fait partie de debhelper.
