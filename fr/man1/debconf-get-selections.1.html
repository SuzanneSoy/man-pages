NOM

debconf-get-selections - afficher le contenu de la base de données de debconf  

SYNOPSIS

debconf-get-selections [--installer]  

DESCRIPTION

Afficher la base de données actuelle de debconf dans un format compréhensible par debconf-set-selections.

Pour sauvegarder la base de données de l'installateur Debian, depuis /var/log/installer/cdebconf, utilisez le paramètre --installer.
