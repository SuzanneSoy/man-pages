NOM

dh_lintian - Installer les fichiers « override » de lintian dans le répertoire de construction du paquet  

SYNOPSIS

dh_lintian [options_de_debhelper]  

DESCRIPTION

dh_lintian est le programme de la suite debhelper chargé de l'installation, dans le répertoire de construction du paquet, des fichiers « override » utilisés par lintian.  

FICHIERS

debian/paquet.lintian-overrides
Installé dans le répertoire de construction du paquet, sous usr/share/lintian/overrides/paquet. Ce fichier est utilisé pour supprimer les diagnostics erronés de lintian.
debian/source/lintian-overrides
Ces fichiers ne sont pas installés, mais seront pris en compte par lintian pour induire des modifications de comportement (overrides) pour le paquet source.
