NOM

dh_installinfo - Installer les fichiers info  

SYNOPSIS

dh_installinfo [options_de_debhelper] [-A] [fichier ...]  

DESCRIPTION

dh_installinfo est le programme de la suite debhelper chargé de l'installation des fichiers info dans usr/share/info dans le répertoire de construction du paquet.

Depuis la version 11 de debhelper, dh_install cherchera dans l'arborescence debian/tmp pour trouver les fichiers s'il ne les trouve pas dans le répertoire courant (ou dans celui indiqué par l'utilisation de --sourcedir).  

FICHIERS

debian/paquet.info
Liste les fichiers info à installer.

Supports substitution variables in compat 13 and later as documented in debhelper(7).

 

OPTIONS

-A, --all
Installe l'ensemble des fichiers indiqués sur la ligne de commande dans tous les paquets construits.
--sourcedir=répertoire
Cherche dans le répertoire indiqué les fichiers à installer. Cette option nécessite le niveau de compatibilité 11 ou supérieur (et sera silencieusement ignorée en version 10 et avant).

Nota : Cette option ne fait pas la même chose que --sourcedirectory utilisée par dh_auto_*. Il est rare d'avoir besoin d'utiliser cette option puisque dh_installinfo cherche automatiquement les fichiers dans debian/tmp depuis le niveau de compatibilité 11 de debhelper.

fichier ...
Installe les fichiers info dans le premier paquet construit (ou dans tous les paquets si -A est indiqué).
 

VOIR AUSSI

debhelper(7)

Ce programme fait partie de debhelper.
