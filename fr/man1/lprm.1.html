NOM

lprm - Annuler des travaux d'impression.  

SYNOPSIS

lprm [ -E ] [ -U identifiant ] [ -h serveur[:port] ] [ -P destination[/instance] ] [ - ] [ identifiant(s)_travaux ]  

DESCRIPTION

lprm annule des travaux d'impression en attente. Si aucun paramètre n'est fourni, c'est le travail en cours sur la destination par défaut qui est annulé. Il est possible de fournir un ou plusieurs identifiants de travaux d'impression à annuler ou d'utiliser l'option - pour annuler tous les travaux.  

OPTIONS

lprm accepte les options suivantes :
-E
Obliger un chiffrement lors de la connexion au serveur.
-P destination[/instance]
Indiquer l'imprimante ou la classe d'imprimante de destination.
-U identifiant
Utiliser un autre identifiant.
-h serveur[:port]
Indiquer un serveur alternatif.
 

CONFORMITÉ

La version CUPS de lprm est compatible avec la version standard de Berkeley.  

EXEMPLES

Annuler le travail en cours sur l’imprimante par défaut : 

    lprm
Annuler le travail 1234 : 

    lprm 1234
Annuler tous les travaux d'impression : 

    lprm -
