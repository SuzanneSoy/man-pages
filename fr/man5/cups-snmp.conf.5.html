NOM

snmp.conf - snmp configuration file for cups (deprecated)  

DESCRIPTION

Le fichier snmp.conf configure la façon dont les programmes de transmission réseau de CUPS (http, https, ipp, ipps, lpd, snmp et socket) accèdent aux informations de l'imprimante en utilisant SNMP version 1 et se trouve normalement dans le répertoire /etc/cups. Chaque ligne du fichier est une directive de configuration, une ligne vide ou un commentaire. Les commentaires commencent par le caractère « # ».

Les directives Community et DebugLevel sont utilisées par tous les programmes de transmission. Le reste ne s'applique qu'au programme de transmission SNMP (cups-snmp(8)).  

DIRECTIVES

Les directives suivantes sont comprises par les programmes de transmission réseau :
Address @IF(nom)
Address @LOCAL
Address adresse
Émettre les requêtes SNMP (pour recherche) vers la ou les adresses indiquées. Il n'y a pas de valeur par défaut pour l'adresse d'émission.
Community nom
Specifies the community name to use. Only a single community name may be specified. The default community name is "public". If no name is specified, all SNMP functions are disabled.
DebugLevel nombre
Indiquer le niveau de journalisation de 0 (aucun) à 3 (tout). Généralement cela n'est utilisé qu'à des fins de débogage (d'où le nom). Le niveau par défaut de débogage est 0.
DeviceURI "exp_rationnelle" uri_périphérique [... uri_périphérique]
Indiquer un ou plusieurs URI de périphériques qui doivent être utilisés pour une marque et un modèle. L'expression rationnelle est utilisée pour correspondance avec la marque et le modèle, et les chaînes d’URI du périphérique doivent être de la forme « schéma://%s[:port]/[chemin] », où « %s » représente l'adresse détectée ou le nom d'hôte. Il n'y a pas de règle d'harmonisation pour les schémas d’URI de périphériques.
HostNameLookups on
HostNameLookups off
Définir si les adresses des imprimantes doivent être converties en noms d'hôte ou doivent être laissées en adresses IP numériques. La valeur par défaut est « off ».
MaxRunTime secondes
Indiquer le nombre maximal de secondes entre les requêtes du programme de transmission SNMP pour la découverte des imprimantes réseau. La valeur par défaut est de 120 secondes (2 minutes).
 

NOTES

CUPS backends are deprecated and will no longer be supported in a future feature release of CUPS. Printers that do not support IPP can be supported using applications such as ippeveprinter(1).
