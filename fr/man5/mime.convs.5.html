NOM

mime.convs - mime type conversion file for cups (deprecated)  

DESCRIPTION

Le fichier mime.convs définit les filtres disponibles pour convertir des fichiers d'un format vers un autre. Les filtres standards concernent les fichiers textes, PDF, PostScript, ainsi que de nombreux formats graphiques.

Des filtres supplémentaires peuvent être ajoutés avec l’extension .convs dans le répertoire de configuration de CUPS.

Chaque ligne du fichier mime.convs est un commentaire, une ligne vide ou une définition de filtre. Les commentaires commencent par le caractère « # ». Les lignes contenant un filtre indiquent les types MIME source et cible, le coût virtuel du filtre, puis le filtre lui-même : 


    source/type destination/type coût filtre
Le champ source/type précise le type MIME du média source utilisé par le filtre.

Le champ destination/type précise le type MIME du média de destination produit par le filtre.

Le champ coût précise le coût relatif d’exécution du filtre. Une valeur de 100 signifie que le filtre emploie une ressource importante, et 0 une très petite quantité.

Le champ filtre précise le nom de fichier du programme de filtre. Les noms de fichier sont en rapport avec le répertoire des filtres CUPS.  

FICHIERS

/etc/cups — répertoire de configuration classique de CUPS.
/usr/lib/cups/filter — répertoire de configuration classique des filtres CUPS.
/usr/libexec/cups/filter - répertoire de filtres CUPS pour macOS.  

EXEMPLES

Définir un filtre convertissant des documents PostScript au format Raster de CUPS : 

    application/vnd.cups-postscript application/vnd.cups-raster 50 pstoraster
 

NOTES

CUPS filters are deprecated and will no longer be supported in a future feature release of CUPS. Printers that do not support IPP can be supported using applications such as ippeveprinter(1).
