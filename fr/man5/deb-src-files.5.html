NOM

deb-src-files - Format des fichiers distribués de Debian  

SYNOPSIS

debian/files  

DESCRIPTION

Ce fichier fournit la liste des objets qui seront distribués au moyen du fichier de contrôle .changes.

Les éléments du fichier debian/files sont séparés simplement par des espaces.

nom-de-fichier section priorité [ mot-clé=valeur... ]

nom-de-fichier est le nom de l'objet à distribuer.

section et priority correspondent respectivement aux champs de contrôle présents dans le fichier .deb. Les valeurs autorisées sont spécifiques pour chaque distribution.

mot-clé=valeur... correspond à une liste optionnelle des attributs séparés par des espaces pour cette entrée. Le seul mot clé pris en charge actuellement est automatic avec la valeur yes, pour marquer les fichiers générés automatiquement.  

NOTES

Ce fichier n'est pas censé être modifié directement, veuillez utiliser soit dpkg-gencontrol ou dpkg-distaddfile pour lui ajouter des entrées.  

VOIR AUSSI

dpkg-genchanges(1), dpkg-distaddfile(1).
