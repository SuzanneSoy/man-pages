NOM

cupsd-helper - cupsd helper programs (deprecated)  

SYNOPSIS

cups-deviced id_requête limite identifiant options
cups-driverd cat nom_ppd
cups-driverd list id_requête limite options
cups-exec sandbox-profile [ -g groupe ] [ -n valeur_nice ] [ -u identifiant ] /chemin/vers/programme argv0 ... argvN  

DESCRIPTION

Le programme cupsd-helper réalise des opérations longues pour le compte de l’ordonnanceur, cupsd(8). Le programme assistant cups-deviced exécute chaque backend(7) de CUPS sans argument pour découvrir les imprimantes disponibles.

Le programme assistant cups-driverd liste toutes les imprimantes disponibles, un sous-ensemble de pilotes d’imprimante « correspondants », ou une copie d’un fichier de pilote PPD particulier.

Le programme assistant cups-exec exécute les programmes de transmission, les filtres et d’autres programmes. Sur macOS, ces programmes sont exécutés dans un bac à sable (« sandbox »).  

FICHIERS

Le programme assistant cups-driverd recherche les fichiers PPD ou d’information dans les répertoires suivants : 

    /Library/Printers
    /opt/share/ppd
    /System/Library/Printers
    /usr/local/share/ppd
    /usr/share/cups/drv
    /usr/share/cups/model
    /usr/share/ppd

Les fichiers PPD peuvent être compressés avec le programme gzip(1) ou incorporés dans une archive tar(1) compressée pour diminuer leur taille.

Les fichiers d’information sur les pilotes doivent se conformer au format indiqué dans ppdcfile(5).  

NOTES

CUPS printer drivers, backends, and PPD files are deprecated and will no longer be supported in a future feature release of CUPS. Printers that do not support IPP can be supported using applications such as ippeveprinter(1).
