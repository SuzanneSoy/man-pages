deb-preinst - pre-installatiescript van de pakketonderhouder  

OVERZICHT

preinst  

BESCHRIJVING

Een pakket kan verschillende pre-installatieacties uitvoeren via scripts van de pakketonderhouder door in zijn control-archief een uitvoerbaar bestand preinst op te nemen (d.w.z. DEBIAN/preinst bij het creëren van het pakket).

Het script kan op de volgende manieren geactiveerd worden:

new-preinst install
Vooraleer het pakket geïnstalleerd wordt.
new-preinst install oude-versie nieuwe-versie
Vooraleer een verwijderd pakket opgewaardeerd wordt.
new-preinst upgrade oude-versie nieuwe-versie
Vooraleer het pakket opgewaardeerd wordt.
old-preinst abort-upgrade nieuwe-versie
Indien postrm mislukt tijdens het opwaarderen of mislukt bij een mislukte opwaardering.
