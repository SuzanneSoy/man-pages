dh_installxfonts - regista fonts X  

RESUMO

dh_installxfonts [debhelper opções]  

DESCRIÇÃO

dh_installxfonts é um programa debhelper que é responsável por registar tipos de letra (fonts) do X, para que os seus correspondentes fonts.dir, fonts.alias, e fonts.scale sejam compilados correctamente durante a instalação.

Antes de chamar este programa, você deve ter instalado quaisquer fonts X disponibilizadas pelo seu pacote na localização apropriada no directório de compilação do pacote, e você tem ficheiros fonts.alias ou fonts.scale, você deve instalá-los na localização correcta sob etc/X11/fonts no seu directório de compilação do pacote.

O seu pacote deve depender de xfonts-utils para que os comandos update-fonts-* estejam disponíveis. (Este programa adiciona a dependência a ${misc:Depends}.)

Este programa gera automaticamente os comandos postinst e postrm necessários para registar fonts do X. Estes comandos são inseridos nos scripts do mantenedor pelo dh_installdeb. Veja dh_installdeb(1) para uma explicação sobre como isto funciona.  

NOTAS

Veja update-fonts-alias(8), update-fonts-scale(8), e update-fonts-dir(8) para mais informação acerca de instalação de fonts X.

Veja Debian policy, secção 11.8.5. para detalhes acerca de fazer fonts à maneira de Debian.
