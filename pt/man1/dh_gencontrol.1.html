NOME

dh_gencontrol - gera e instala ficheiro de controle  

RESUMO

dh_gencontrol [debhelper opções] [-- params]  

DESCRIÇÃO

dh_gencontrol é um programa debhelper que é responsável por gerar ficheiros de controle, e instalá-los no directório DEBIAN com as permissões apropriadas.

Este programa é meramente um wrapper em volta de dpkg-gencontrol(1), o qual o chama uma vez por cada pacote em que actua (mais os pacotes dbgsym relacionados), e passa para ele algumas bandeiras adicionais úteis.

Note que se você usar dh_gencontrol, você também tem de usar dh_builddeb(1) para compilar os pacotes. Caso contrário, a sua compilação pode falhar pois o dh_gencontrol (via dpkg-gencontrol(1)) declara quais pacotes são compilados. Como o debhelper gera automaticamente pacotes dbgsym, por vezes adiciona pacotes adicionais, que serão compilados por dh_builddeb(1).  

OPÇÕES

-- params
Passa params para dpkg-gencontrol(1).
-uparams, --dpkg-gencontrol-params=params
Este é outro modo de passar params para dpkg-gencontrol(1). Está descontinuado, use -- em vez deste.
