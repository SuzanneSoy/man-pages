NOME

dh_clean - limpa os directórios de compilação de pacotes  

RESUMO

dh_clean [debhelper options] [-k] [-d] [-Xitem] [path ...]  

DESCRIÇÃO

dh_clean é um programa debhelper que é responsável pela limpeza após um pacote ser compilado. Remove os directórios de compilação do pacote, e remove mais alguns ficheiros incluindo debian/files, e quaisquer detritos deixados por outros comandos debhelper. Também remove ficheiros comuns que não deveriam aparecer num diff de Debian:
  #*# *~ DEADJOE *.orig *.rej *.SUMS TAGS .deps/* *.P *-stamp

Não corre o ``make clean'' para limpara após o processo de compilação. Use dh_auto_clean(1) para fazer as coisas dessa maneira.

dh_clean deve ser o último comando debhelper a correr na meta clean em debian/rules.  

FICHEIROS

debian/clean
Pode listar outros caminhos para serem removidos.

Note que os directórios listados neste ficheiro devem terminar com uma barra final. Qualquer conteúdo nestes directórios será também removido.

Supports substitution variables in compat 13 and later as documented in debhelper(7).

 

OPÇÕES

-k, --keep
Isto está descontinuado, use dh_prep(1) em vez deste.

A opção foi removida no nível de compatibilidade 12.

-d, --dirs-only
Apenas limpa os directórios de compilação do pacote, não limpa mais nenhuns outros ficheiros.
-Xitem --exclude=item
Exclui ficheiros que contenham item em qualquer ponto do seu nome de ficheiro de serem apagados, mesmo se estes fossem normalmente apagados. Você pode usar esta opção várias vezes para construir uma lista de coisas a excluir.
path ...
Apaga estes caminhos também.

Note que os directórios passados como argumentos devem terminar com uma barra final. Qualquer conteúdo nestes directórios será também removido.
