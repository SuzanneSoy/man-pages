NOME

dh_md5sums - gera o ficheiro DEBIAN/md5sums  

RESUMO

dh_md5sums [opções do debhelper] [-x] [-Xitem] [--include-conffiles]  

DESCRIÇÃO

dh_md5sums é um programa debhelper que é responsável por gerar um ficheiro DEBIAN/md5sums o qual lista os md5sums de cada ficheiro no pacote. Estes ficheiros são usados por dpkg --verify ou pelo programa debsums(1).

Todos os ficheiros em DEBIAN/ são omitidos do ficheiro md5sums, pois eles são todos ficheiros de configuração (a menos que você use o switch --include-conffiles).

O ficheiro md5sums é instalado com o dono e permissões apropriadas.  

OPÇÕES

-x, --include-conffiles
Inclui ficheiros de configuração na lista de md5sums. Note que esta informação é redundante pois isso está incluído noutro local nos pacotes Debian.
-Xitem, --exclude=item
Exclui ficheiros que contêm item em qualquer ponto do seu nome de ficheiro de serem listados no ficheiro md5sums.
