NOME

dh_installchangelogs - instala relatórios de alterações (changelogs) em directórios de compilação de pacotes.  

RESUMO

dh_installchangelogs [debhelper opções] [-k] [-Xitem] [upstream]  

DESCRIÇÃO

dh_installchangelogs é um programa debhelper que é responsável por instalar relatórios de alterações (changelogs) nos directórios de compilação de pacotes.

An upstream changelog file may be specified as an option.

Se existir um ficheiro changelog do autor, este será instalado como usr/share/doc/package/changelog no directório de compilação do pacote.

Se o relatório de alterações especificado do autor for um ficheiro html (determinado pela extensão do ficheiro), em vez disso será instalado como usr/share/doc/package/changelog.html. Se o relatório html for convertido para texto simples, essa variante pode ser especificada como o segundo ficheiro de relatório de alterações do autor. Quando nenhuma variante de texto simples é especificada, é gerado um curto usr/share/doc/package/changelog, apontando os leitores para o ficheiro de relatório html.  

FICHEIROS

debian/changelog
debian/NEWS
debian/pacote.changelog
debian/pacote.NEWS
Instalado automaticamente em usr/share/doc/pacote/ no directório de compilação do pacote.

Usa o nome específico do pacote se o pacote precisar de um ficheiro NEWS ou changelog diferente.

O ficheiro changelog é instalado com o nome changelog para os pacotes nativos, e changelog.Debian para pacotes não-nativos. O ficheiro de NOTICIAS é sempre instalado com o nome NEWS.Debian.

 

OPÇÕES

-k, --keep
Mantêm o nome original do registo de alterações do autor. Isto será conseguido ao instalar o registo de alterações do autor como changelog, e criando um link simbólico daí para o nome original do ficheiro changelog. Isto pode ser útil se o registo de alterações do autor tiver um nome fora do usual, ou se outra documentação no pacote faça referência ao ficheiro changelog.
-Xitem, --exclude=item
Exclui ficheiros changelog do autor que contenham item em qualquer ponto do seu nome de ficheiro, de serem instalados.

Note que o nome de directório do changelog é também parte correspondente.

upstream
Instala este ficheiro como o registo de alterações (changelog) da origem.
