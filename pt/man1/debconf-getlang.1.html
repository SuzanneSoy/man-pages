NOME

debconf-getlang - extrai uma linguagem a partir de um ficheiro de templates  

SINOPSE

 debconf-getlang lang master [translation]
 debconf-getlang --stats master translation [...]
 

DESCRIÇÃO

Nota: Este utilitário está obsoleto; você deve passar a usar o pacote po-debconf.

Este programa ajuda a criar e a gerir traduções de templates debconf. Basicamente existem três situações nas quais este programa pode ser chamado:

Uma tradução foi agora iniciada.
Você quer fornecer aos tradutores um ficheiro que eles possam trabalhar que tem os campos em Inglês do ficheiro templates, mais campos Field-ll vazios para a linguagem de destino que eles vão preencher.

Para fazer isto, execute o programa com o primeiro parâmetro a ser o código da linguagem para a qual se está a traduzir, e o segundo parâmetro sendo o nome do ficheiro do ficheiro de templates em Inglês.

Uma tradução está em curso.
Você alterou algum texto em Inglês, ou adicionou mais itens ao seu ficheiro templates, e quer enviar aos tradutores um ficheiro com o texto em Inglês mais as suas traduções actuais (ou você é o tradutor e quer gerar tal ficheiro para seu próprio uso).

Para fazer isto, execute o programa com o primeiro parâmetro a ser o código da linguagem para a qual se está a traduzir, o segundo parâmetro sendo o nome do ficheiro do ficheiro de templates mestre em Inglês, e o terceiro parâmetro sendo o nome do ficheiro traduzido actual.

Quando corre deste modo, o programa é suficientemente inteligente para detectar traduções fuzzy (aproximadas). Por exemplo, uma descrição fuzzy será mostrada como Description-<lang>-fuzzy, e será adicionado uma nova Description-<lang> em branco. Os tradutores devem remover os campos -fuzzy e corrigir as traduções marcadas com fuzzy.

A verificar o estado duma tradução
Para verificar o estado duma tradução, use a bandeira --status, e passe o ficheiro template Inglês como o primeiro parâmetro, e todos os outros templates traduzidos depois disso. Irá gerar estatísticas para cada um deles. Por exemplo: 

  debconf-getlang --stats debian/templates debian/templates.*
 

NOTA

Note que o texto nos templates gerados pode ter as linhas arrumadas relativamente a palavras pelo debconf.
