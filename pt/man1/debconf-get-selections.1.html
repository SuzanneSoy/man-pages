NOME

debconf-get-selections - escreve o conteúdo da base de dados do debconf  

SINOPSE

debconf-get-selections [--installer]  

DESCRIÇÃO

Gera a base de dados debconf actual num formato compreensível por debconf-set-selections.

Para abandonar a base de dados debconf do debian-installer a partir de /var/log/installer/cdebconf, use o parâmetro --installer.
