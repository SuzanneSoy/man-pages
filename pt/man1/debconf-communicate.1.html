NOME

debconf-communicate - comunicar com o debconf  

SINOPSE

 echo comandos | debconf-communicate [opções] [pacote]
 

DESCRIÇÃO

debconf-communicate permite-lhe comunicar com o debconf na hora, a partir da linha de comandos. O argumento package é o nome do pacote o qual você finge ser quando comunica com o debconf, e pode ser omitido se você for preguiçoso. Lê comandos no formato usado pelo protocolo debconf a partir do stdin. Para documentação sobre os comandos disponíveis e a sua utilização, veja a especificação debconf.

Os comandos são executados em sequência. O código de retorno textual de cada é escrito na saída standard.

O valor devolvido deste programa é o código de retorno numérico do último comando executado.  

EXEMPLO

 echo get debconf/frontend | debconf-communicate

Escreve o valor da questão debconf/frontend.  

AVISO

Este programa nunca deve ser usado a partir dum script de responsável de um pacote que use debconf! Pode, no entanto, ser útil em depuração.
