NOME

dh_installinitramfs - instala hooks de initramfs e scripts principais de configuração  

RESUMO

dh_installinitramfs [opções do debhelper] [-n]  

DESCRIÇÃO

dh_installinitramfs é um programa debhelper que é responsável por instalar um pacote Debian que fornece hooks de initramfs

Se o dh_installinitramfs instalar ou (em compatibilidade 12 ou posterior) detectar um ou mais hooks de initramfs no pacote, então também gera automaticamente os comandos postinst e postrm necessários para interagir com o sistema initramfs de Debian menu. Estes comandos são inseridos nos scripts do mantenedor pelo dh_installdeb(1).  

FICHEIROS

debian/pacote.initramfs-hook
Assumido ser um hook de initramfs que será instalado em usr/share/initramfs-tools/hooks/package no directório de compilação do pacote. Veja HOOK SCRIPTS em initramfs-tools(8) para mais informação acerca de hooks de initramfs.
 

OPÇÕES

-n, --no-scripts
Não modifique os scripts postinst/postrm.
 

NOTAS

Note que este comando não é idempotente. O dh_prep(1) deve ser chamado entre invocações deste comando. Caso contrário, pode causar múltiplas instâncias do mesmo texto a ser adicionado aos scripts do mantenedor.  

VEJA TAMBÉM
