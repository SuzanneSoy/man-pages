NOME

dh_auto_test - corre automaticamente os conjuntos de testes dum pacote  

RESUMO

dh_auto_test [opções do sistema de compilação] [opções do debhelper] [-- parâmetros]  

DESCRIÇÃO

dh_auto_test é um programa debhelper que tenta correr automaticamente uma suite de teste de um pacote. Fá-lo ao correr o comando apropriado para o sistema de compilação que detecta que o pacote usa. Por exemplo, Se existir um Makefile e conter uma meta test ou check, então é feito ao correr make (ou MAKE, se a variável de ambiente estiver definida). Se a suite de teste falhar, o comando irá terminar com não-zero. Se não existir uma suite de teste, irá terminar com zero sem fazer nada.

Isto entende-se que deve funcionar com cerca de 90% dos pacotes numa suite de teste. Se não funcionar, você é encorajado não usar o dh_auto_test, e correr a suite de teste manualmente.  

OPÇÕES

Veja "BUILD SYSTEM OPTIONS" in debhelper(7) para uma lista de selecção de sistemas de compilação comuns e opções de controle.
-- params
Passa params para o programa que é executado, após os parâmetros que o dh_auto_test normalmente passa.
 

NOTAS

Se a variável de ambiente DEB_BUILD_OPTIONS conter nocheck, nenhum teste será executado.
