NOME

accessdb - despeja o conteúdo de uma base de dados man-db num formato legível  

SINOPSE

/usr/sbin/accessdb [-d?V] [<index-file>]  

DESCRIÇÃO

accessdb imprime os dados contidos dentro de uma base de dados man-db em formato legível. Por predefinição, despeja os dados de /var/cache/man/index.<db-type>, onde <db-type> depende da biblioteca de base de dados em uso.

Fornecer um argumento a accessdb sobrepõe esta predefinição.  

OPÇÕES

-d, --debug
imprime informação de depuração.
-?, --help
mostra uma mensagem de ajuda e sai.
--usage
mostra uma mensagem curta de uso e sai.
-V, --version
mostra informação da versão.
