NOME

debconf-loadtemplate - carrega um arquido de modelo em uma base de dados debconf  

RESUMO

 debconf-loadtemplate dono arquivo [arquivo ..]
 

DESCRIÇÃO

Loads one or more template files into the debconf database. The first parameter specifies the owner of the templates (typically, the owner is the name of a debian package). The remaining parameters are template files to load.  

AVISO

Este programa não deverá nunca ser usado em um script de mantenedor de um pacote que usa debconf! Ele pode, porém, ser útil em depuração ou para alimentar a base de dados debconf.  

VEJA TAMBÉM
