NAME

vipw - 패스워드 파일 편집  

사용법

vipw  

설명

Vipw 명령은 /etc/passwd 파일을 잠그고(lock) 편집하고, 편집이 끝나면, /etc/passwd 파일을 잠금해제(unlock)한다. 이미 /etc/passwd 파일이 잠겨 있으면, 다음에 사용하라고 메시지를 보여준다. vipw에서 사용하는 기본 편집기는 vi(1)이다.  

환경변수

vipw는 다음 환경 변수를 사용한다.

EDITOR
사용할 편집기 지정. 초기값은 vi

 

관련 항목

passwd(1), vi(1), passwd(5)  

발자취

명령은 BSD 4.0 에서 처음 나타났다.
