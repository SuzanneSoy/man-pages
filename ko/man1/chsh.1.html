NAME

chsh - 사용자 로그인 쉘을 바꾼다.  

SYNOPSIS

chsh [ -s shell ] [ -l ] [ -u ] [ -v ] [ username ]  

DESCRIPTION

chsh 명령은 사용자가 사용하고 있는 로그인 쉘을 바꾸는데, 사용된다. (이말은 exit 명령을 사용했을 때 로그아웃되는 쉘을 말한다.) 명령행에서 쉘을 지정하지 않으면, 사용자에게 물어본다.  

VALID SHELLS

chsh 명령에서는 절대 경로를 지정하는 어떠한 실행파일도 지정이 가능하다. 하지만, /etc/shells 파일에서 지정되어 있지 않는 풀그림을 사용할 경우에는 경고 메시지를 보여준다.  

OPTIONS

-s, --shell
지정하는 쉘을 앞으로 사용할 로그인 쉘로 바꾼다.
-l, --list-shells
/etc/shells 파일 안에 지정된 쉘을 나열하고 마친다.
-u, --help
도움말을 보여주고 마친다.
-v, --version
버전 정보를 보여주고 마친다.
