НАЗВАНИЕ

zsoelim - выполняет .so запросы для входящих данных roff  

СИНТАКСИС

zsoelim [-CVh] [файл ...]  

ОПИСАНИЕ

zsoelim анализирует параметры файл или, если ничего не указано, строки из стандартного входного потока в виде:

.so <имя файла>

Эти запросы заменяются содержимым указанного файла. Если запрос не может быть удовлетворён, то zsoelim ищет имя_файла.расширение, где .расширение может быть одним из .gz, .Z или .z. Могут поддерживаться другие типы расширений в зависимости от параметров компиляции программы. Если запрос решён с помощью сжатого файла, то данный файл расжимается с помощью соответствующей программы, и результат используется для удовлетворения запроса.

Обычно, soelim-программы используются, чтобы позволить препроцессорам roff обрабатывать файлы, указанные в запросах. Данная версия была написана, чтобы обойти проблемы, созданные поддержкой сжатых справочных страниц.  

ПАРАМЕТРЫ

-C, --compatible
Данный параметр введён для совместимости с другими программами soelim. Он позволяет использовать для отделения имени файла в .so запросах символы, отличные от пробелов. Так как это является поведением по умолчанию, то данный параметр игнорируется.
-h, --help
Показать справочное сообщение и закончить работу.
-V, --version
Показать информацию о версии.
