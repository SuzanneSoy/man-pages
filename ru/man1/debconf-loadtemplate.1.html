НАЗВАНИЕ

debconf-loadtemplate - загружает файл шаблона в базу данных debconf  

СИНТАКСИС

 debconf-loadtemplate владелец файл [файл ..]
 

ОПИСАНИЕ

Загружает один или более файлов шаблонов в базу данных debconf. Первый параметр указывает владельца шаблонов (обычно, владелец это имя пакета debian). Остальные параметры воспринимаются как файлы для загрузки.  

ПРЕДУПРЕЖДЕНИЕ

Эта программа никогда не должна использоваться из сопровождающего сценария пакета, который использует debconf! Однако, она может оказаться полезной для отладки или заполнения базы данных debconf.
