НАЗВАНИЕ

expiry - проверяет и изменяет пароль согласно политике устаревания  

СИНТАКСИС

expiry параметр
 

ОПИСАНИЕ

The expiry command checks (-c) the current password expiration and forces (-f) changes when required. It is callable as a normal user command.  

ПАРАМЕТРЫ

The options which apply to the expiry command are:

-c, --check

Проверяет и изменяет срок действия пароля у текущего пользователя.

-f, --force

Принудительно меняет пароль, если его срок действия истёк.

-h, --help

Показать краткую справку и закончить работу.
