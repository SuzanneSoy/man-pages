НАЗВАНИЕ

debconf-get-selections - выводит содержимое базы данных debconf  

СИНТАКСИС

debconf-get-selections [--installer]  

ОПИСАНИЕ

Выводит содержимое базы данных debconf в формате, который понимает команда debconf-set-selections.

Чтобы получить данные по настройке программы установки debian-installer базы данных debconf из /var/log/installer/cdebconf используйте параметр --installer.
