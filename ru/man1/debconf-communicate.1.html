НАЗВАНИЕ

debconf-communicate - программа для взаимодействия с debconf  

СИНТАКСИС

 echo команды | debconf-communicate [параметры] [пакет]
 

ОПИСАНИЕ

debconf-communicate позволяет обращаться к debconf из командной строки. В необязательном параметре «пакет» указывается название пакета от имени которого производится обращение к debconf. debconf-communicate читает команды из стандартного входного потока (stdin), которые задаются формате протокола команд debconf. Описание допустимых команд и их использование можно найти в спецификации к debconf.

Команды выполняются последовательно одна за другой. Результат работы каждой из них записывается в текстовом виде в стандартный выходной поток.

Данная программа возвращает значение равное числовому коду завершения последней выполненной команды.  

ПРИМЕР

 echo get debconf/frontend | debconf-communicate

Показывает сохранённый ответ на вопрос о debconf/frontend.  

ПРЕДУПРЕЖДЕНИЕ

Эта программа никогда не должна вызываться из сценариев сопровождения пакета, который использует debconf! Однако она может пригодиться при отладке.
