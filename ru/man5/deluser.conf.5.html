NAME

/etc/deluser.conf - файл настройки для deluser(8) и delgroup(8).  

ОПИСАНИЕ

Файл /etc/deluser.conf содержит установки по умолчанию для программ deluser(8) и delgroup(8). В каждой строке содержится по одному значение в формате параметр = значение. Значение можно помещать в двойные или одинарные кавычки. Строки комментария должны иметь знак фунта (#) в начале строки.

Команды deluser(8) и delgroup(8) также читают /etc/adduser.conf, смотрите adduser.conf(5); настройки в deluser.conf могут быть изменены настройками из adduser.conf.

Действующие параметры настройки:

REMOVE_HOME
Удалять домашний каталог и хранилище почты удаляемого пользователя. Значениями могут быть 0 (не удалять) или 1 (удалять).
REMOVE_ALL_FILES
Стирать все файлы в системе, принадлежащие удаляемому пользователю. Если этот параметр включён, то REMOVE_HOME бесполезен. Значениями могут быть 0 или 1.
BACKUP
Если REMOVE_HOME или REMOVE_ALL_FILES включены, то перед удалением делается резервная копия всех файлов. Будет создан архивный файл с именем имя_пользователя.tar(.gz|.bz2) в каталоге, указанном в BACKUP_TO. Из возможных методов сжатия выбирается лучший. Значениями могут быть 0 или 1.
BACKUP_TO
Если BACKUP включён, то в BACKUP_TO указывается каталог, куда будет производиться резервное копирование. По умолчанию используется текущий каталог.
NO_DEL_PATHS
Список регулярных выражений разделённых пробелами. Все файлы, которые будут удаляться в случае удаления домашнего каталога или принадлежащие пользователю, сначала сравниваются с этими регулярными выражениями. Если обнаруживается совпадение, то файл не удаляется. По умолчанию в список входят системные каталоги, остаётся только /home.

Другими словами: по умолчанию только файлы внутри /home, принадлежащие заданному пользователю, будут удаляться.

ONLY_IF_EMPTY
Удалять группу, только если в ней нет пользователей. По умолчанию 0.
EXCLUDE_FSTYPES
Регулярное выражение, описывающее файловые системы, которые нужно исключить из поиска при удалении файлов пользователя. По умолчанию "(proc|sysfs|usbfs|devpts|tmpfs|afs)".
