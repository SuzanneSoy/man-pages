НАЗВАНИЕ

dpkg-preconfigure - позволяет пакетам задать вопросы перед установкой  

СИНТАКСИС

 dpkg-preconfigure [параметры] пакет.deb

 dpkg-preconfigure --apt
 

ОПИСАНИЕ

dpkg-preconfigure позволяет пакетам задать вопросы перед установкой. Она работает с пакетами debian и всеми пакетами, которые будут использовать debconf в своих сценариях config для проверки системы и задания вопросов.  

ПАРАМЕТРЫ

-fинтерфейс, --frontend=интерфейс
Указать нужную интерфейсную программу.
-pзначение_приоритета, --priority=значение_приоритета
Установить низший приоритет задаваемых вопросов. Все вопросы с приоритетом ниже указанного будут игнорироваться, а в качестве ответов будут использоваться ответы по умолчанию.
--terse
Включить краткий режим вывода. Влияет только на некоторые интерфейсные программы.
--apt
Работать в режиме apt. Вместо получения списка имён файлов пакетов из параметров он считывается со стандартного входного потока (stdin). Обычно этот режим используется apt, которая запускает dpkg-preconfigure для всех пакетов перед их установкой. Для этого добавьте в /etc/apt/apt.conf: 

 // Предварительная настройка всех пакетов перед
 // их установкой.
 DPkg::Pre-Install-Pkgs {
        "dpkg-preconfigure --apt --priority=low";
 };
-h, --help
Показать справку по использованию.
