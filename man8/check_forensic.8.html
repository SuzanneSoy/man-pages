NAME

check_forensic - tool to extract mod_log_forensic output from apache log files  

SYNOPSIS

check_forensic <log_file>  

DESCRIPTION

chech_forensic is a simple shell script designed to help apache administrators to extract mod_log_forensic output from apache2 log files. It checks the forensic log for requests that did not complete and outputs the request log for each one.
