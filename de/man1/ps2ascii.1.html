NAME

ps2ascii - Ghostscript-Übersetzer von PostScript oder PDF nach ASCII  

SYNTAX

ps2ascii [ Eingabe.ps [ Ausgabe.txt ] ]
ps2ascii Eingabe.pdf [ Ausgabe.txt ]  

BESCHREIBUNG

ps2ascii benutzt gs(1), um ASCII-Text aus einer PostScript(tm)- oder Adobe Portable-Document-Format-(PDF-)Datei zu extrahieren. Falls keine Dateien in der Kommandozeile angegeben werden, liest gs von der Standardeingabe; bei PDF-Dateien wird ein expliziter Dateiname benötigt, keine Standardeingabe. Falls keine Ausgabedatei angegeben ist, wird zur Standardausgabe geschrieben.

ps2ascii ignoriert die Schriftkodierung und kommt nicht sonderlich gut mit Unterscheidung (Kerning) zurecht. Somit sollten Sie für PostScript (aber gegenwärtig nicht für PDF) pstotext benutzen (siehe unten).  

DATEIEN

Starten Sie "gs -h" um den Ort der Ghostscript-Dokumentation auf Ihrem System zu ermitteln, wo Sie weitere Datails finden.  

SIEHE AUCH

pstotext(1), http://www.research.digital.com/SRC/virtualpaper/pstotext.html  

VERSION

Dies Dokument wurde zuletzt für Ghostscript version 7.21 durchgesehen.
