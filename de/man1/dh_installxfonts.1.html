NAME

dh_installxfonts - registriert X-Schriften  

ÜBERSICHT

dh_installxfonts [Debhelper-Optionen]  

BESCHREIBUNG

dh_installxfonts ist ein Debhelper-Programm, das für das Registrieren der X-Schriften zuständig ist, weswegen ihre entsprechenden fonts.dir, fonts.alias und fonts.scale zur Installationszeit neu gebaut werden.

Bevor dieses Programm aufgerufen wird, sollten Sie jegliche X-Schriften, die von Ihrem Paket bereitgestellt werden, an die geeignete Stelle im Paketbauverzeichnis installiert haben. Falls Sie fonts.alias- oder fonts.scale-Dateien haben, sollten Sie sie an die korrekte Stelle unter etc/X11/fonts in Ihrem Paketbauverzeichnis installieren.

Ihr Paket sollte von xfonts-utils abhängen, so dass die update-fonts-*-Befehle verfügbar sind. (Dieses Programm fügt diese Abhängigkeit ${misc:Depends} hinzu.)

Dieses Programm erzeugt automatisch die postinst- und postrm-Befehle, die zum Registrieren von X-Schriften benötigt werden. Diese Befehle werden durch dh_installdeb in die Betreuerskripte eingefügt. Eine Erklärung, wie dies funktioniert, finden Sie in dh_installdeb(1).  

ANMERKUNGEN

Weitere Informationen über die Installation der X-Schriften finden Sie unter update-fonts-alias(8), update-fonts-scale(8) und update-fonts-dir(8).

Einzelheiten über die Debian-Art, mit Schriften umzugehen, finden Sie in der Debian-Richtlinie im Abschnitt 11.8.5.
