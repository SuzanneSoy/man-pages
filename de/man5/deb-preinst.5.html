deb-preinst - Präinstallationsbetreuerskript eines Paketes  

ÜBERSICHT

preinst  

BESCHREIBUNG

Ein Paket kann eine Reihe von Präinstallationsaktionen mittels seiner Betreuerskripte durchführen, indem es eine ausführbare Datei preinst in seinem Steuerarchiv (d.h. DEBIAN/preinst während der Paketerstellung) hinzunimmt.

Das Skript kann in den folgenden Weisen aufgerufen werden:

new-preinst install
Bevor das Paket installiert wird.
new-preinst install alte_Version neue_Version
Bevor ein Upgrade des entfernten Pakets durchgeführt wird.
new-preinst upgrade alte_Version neue_Version
Bevor das Upgrade des Pakets durchgeführt wird.
old-preinst abort-upgrade neue_Version
Falls postrm während des Upgrades oder während eines fehlgeschlagenen Upgrades fehlschlägt.
